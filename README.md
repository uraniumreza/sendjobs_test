## SendJobs (Test)

A basic _react-native_ app along with the state-management tool _react-redux_!

## Run Locally

```
git clone https://github.com/uraniumreza/sendjobs_test.git && cd sendjobs_test
yarn && npm start
react-native run-android
or
react-native run-ios
```
